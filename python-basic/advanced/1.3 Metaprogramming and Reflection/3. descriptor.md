# Descriptor
描述符​ 是实现了特定协议（即拥有 `__get__`, `__set__`, 或 `__delete__`方法中的一个或多个）的类。它允许你自定义在访问一个对象属性时（例如，obj.attr）的实际行为。

简单来说，描述符让你能够控制“点操作符”（.）背后的逻辑。

| 特性 | 元类 (Metaclass) | 描述符 (Descriptor) |
|------|-------------------|-------------------|
| **介入时机** | 类创建时 | 属性访问时 |
| **控制粒度** | 类级别 | 属性级别 |
| **典型应用** | 框架基类、ORM映射、插件系统 | 属性验证、延迟加载、计算属性 |
| **使用频率** | 相对较少（主要用于框架开发） | 相对较多（日常开发中常见） |
| **实现复杂度** | 较高 | 中等 |
| **学习曲线** | 较陡峭 | 相对平缓 |
| **适用对象** | 类的结构和行为 | 单个属性的读写行为 |
| **常见使用场景** | 自动注册子类、添加类属性、修改类定义 | 数据验证、类型检查、属性访问控制 |
| **性能影响** | 主要在类定义时一次性影响 | 在属性访问时持续影响 |


@property, @classmethod, @staticmethod装饰器都是基于描述符协议实现的。
